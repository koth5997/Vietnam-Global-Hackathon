아나콘다 사용권장

pip install nltk spacy pypdf2 matplotlib

주요 기능
PDF 추출: PDF 이력서 파일에서 텍스트 내용을 추출합니다.
맥락 분석 (spaCy): 명명된 개체 인식(NER)을 사용하여 교육, 경력, 수상 등의 관련 섹션을 식별합니다.
키워드 추출 (NLTK): 영화 제작 직무(예: 감독, 영화, 제작)와 관련된 키워드를 포함하는 관련 구문을 추출합니다.
중복 필터링: 문장 내에서 키워드가 중복되어 점수에 여러 번 반영되는 것을 방지합니다.
가중치 부여 점수 계산: 중요도에 따라 키워드에 다른 가중치를 부여합니다.
합격/불합격 기준: 사용자가 설정한 점수 기준에 따라 이력서를 분류합니다.

사용된 기술
Python: 스크립트 개발의 주요 프로그래밍 언어입니다.
NLTK (Natural Language Toolkit): 토큰화, 불용어 제거, 품사 태깅과 같은 텍스트 처리 작업을 위한 라이브러리입니다.
spaCy: 맥락 분석에 사용되는 고급 NLP 라이브러리입니다. 특히 명명된 개체 인식(NER)에 사용됩니다.
PyPDF2: PDF 파일에서 텍스트 내용을 추출하는 라이브러리입니다.

작동 방식
PDF 추출: PyPDF2 라이브러리를 사용하여 제공된 PDF 이력서에서 텍스트 내용을 추출합니다.

맥락 분석: spaCy를 사용하여 명명된 개체 인식(NER)을 기반으로 교육, 경력, 수상 등 관련 섹션을 식별합니다. 이를 통해 관련 키워드가 포함될 가능성이 가장 높은 영역에 분석을 집중할 수 있습니다.

키워드 추출: NLTK는 토큰화, 불용어 제거, 품사 태깅을 수행하여 영화 제작 직무(예: 감독, 영화, 제작)와 관련된 키워드를 포함하는 관련 구문을 식별합니다.

중복 필터링: 집합(set)을 사용하여 문장 내에서 이미 발견된 키워드를 추적합니다. 이렇게 하면 키워드의 첫 번째 발생만 점수에 반영되어 반복되는 키워드로 인해 이력서가 과대 평가되는 것을 방지합니다.

가중치 부여 점수 계산: 각 키워드에는 영화 제작 직무에서의 중요도에 따라 가중치가 부여됩니다. 
예를 들어 "각본가"는 "영화"보다 더 높은 가중치를 가질 수 있습니다. 스크립트는 각 문장을 순회하며 키워드가 발견되면 해당 키워드 가중치를 전체 점수에 더합니다.

합격/불합격 기준: 최종 점수를 사용자가 정의한 기준과 비교합니다. 기준을 초과하는 이력서는 "합격(가능)"으로 분류되고, 기준 미만은 "불합격(불가능)"으로 분류됩니다.

스크립트 사용 방법


키워드 가중치 및 기준 업데이트: relevant_keywords 딕셔너리와 keyword_weights 딕셔너리를 수정하여 관심 있는 키워드와 해당 가중치를 조정할 수 있습니다. 

합격/불합격 기준은 코드 내의 if pdf_score >= 50: 부분을 수정하여 변경할 수 있습니다.

PDF 경로 설정: if __name__ == "__main__": 블록의 pdf_path 변수를 분석할 PDF 이력서 파일의 실제 경로로 업데이트합니다.

스크립트 실행:
 Python 인터프리터를 사용하여 스크립트를 실행합니다. 스크립트는 추출된 텍스트 내용, 식별된 관련 역할, 추출된 키워드 구문, 최종 점수와 함께 합격/불합격 분류를 표시합니다.

주의 사항: 이 스크립트는 기본적인 예제이며 실제 채용 애플리케이션에 사용하려면 추가 개발 및 사용자 정의가 필요할 수 있습니다. 이력서 형식의 다양성과 영화 산업 직무의 특정 요구 사항을 고려하는 것이 중요합니다.
